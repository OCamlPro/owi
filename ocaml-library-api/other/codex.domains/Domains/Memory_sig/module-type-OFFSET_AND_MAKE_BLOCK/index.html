Module type Memory_sig.OFFSET_AND_MAKE_BLOCK

The idea is to match a BLOCK with the representation of an OFFSET within that BLOCK. But we delay the creation of the Block using a Make_Block functor, as the OFFSET can be used to create the VALUE that the BLOCK contains.

module Scalar : Sig.BASE
module Offset : OFFSET with module Scalar = Scalar
module Make_Block (Value : VALUE with module Scalar = Scalar) : BLOCK with module Scalar = Scalar and module Value = Value and module Offset = Offset and type boolean = Value.boolean
