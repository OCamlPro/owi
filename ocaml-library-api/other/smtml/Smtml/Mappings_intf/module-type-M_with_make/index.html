Module type Mappings_intf.M_with_make

The M_with_make module type extends M with a functor for creating instances of M and a flag indicating availability.

module Make () : M

Make () creates a new instance of the M module type.

val is_available : bool

is_available indicates whether the module is available for use.

Will be deprecated in the future, please use Internals.is_available instead.
