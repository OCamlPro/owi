Module type Sig.BITVECTOR_LATTICE_NO_CONVERSION

Bitvector Lattice: a LATTICE representing fixed sized bitvectors. The size isn't stored in the lattice, instead lattice operations take it as an extra argument.

val equal : t -> t -> bool

val compare : t -> t -> int

compare is a total order, and should be compatible with equal.

hash requires that equal values have the same hash.
